Some Info About WebCoreMR



Page Structure:

All of the pages are defined in the xml content folder, with required and optional variables. Required variables include layout, title, and content. The current optional variable is for the menu position.

<page> <layout>oneColumn</layout> <title>About!</title> <menu> <position>2</position> <name>About</name> <key>about</key> </menu> <content> <group>about</group> <file>about</file> <type>html</type> </content> </page>

Navigation:

The navigation bar is generated from the xml content folder, and the menu variable within those xml files. The menu block specifying position, name and key (prevent duplicate position errors). Grouping of links is defined by the folder structure.

<menu> <position>2</position> <name>About</name> <key>about</key> </menu>

CSS & JS

All of the css and js are linked to the pages through xml. These can be added overall (in layout pages) or on a page by page basis (specific files for home, about, etc.).

<cssFiles> <cssFile> <file>base.css</file> <map>base.css.map</map> <group></group> </cssFile> <cssFile> <file>navigation.css</file> <map>navigation.css.map</map> <group></group> </cssFile> </cssFiles>
